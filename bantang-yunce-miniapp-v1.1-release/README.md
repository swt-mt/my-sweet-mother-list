# 半糖孕册 微信小程序正式版 v1.0.0

这是「半糖孕册」第一版微信小程序发布代码，包含完整前台页面、注册登录流程、隐私协议、意见反馈和管理者控制台。

## 页面范围

- 首页：品牌介绍、整体进度、三阶段入口
- 清单页：怀孕必需物品、生产必需物品、新生儿物品清单，共 82 项
- 我的：孕产信息、清单统计、复制准备摘要、恢复默认状态、显示设置、意见反馈入口
- 注册登录页：微信账号/手机号登录流程、头像选择、隐私协议阅读校验
- 隐私协议页：用户信息收集、注册登录、权限调用、本地存储说明
- 意见反馈页：内容纠错、品牌补充、功能建议、使用问题
- 管理者控制台：PV/UV、行为漏斗、反馈处理、版本管理

## 当前版本说明

- 当前是可导入微信开发者工具的完整小程序代码。
- 当前数据使用 `wx.setStorageSync / wx.getStorageSync` 保存在用户本机。
- 当前清单项数：怀孕 18 项、生产 25 项、新生儿 39 项，总计 82 项。
- 当前未接入云开发或服务器，因此多设备同步、真实短信验证码、真实后台统计需要后续接入后端。
- 管理者控制台演示密码：`bantang2026`。
- 版本管理支持管理者手动填写版本号、发版时间和更新内容，并保存到本地。

## 发布前必须处理

1. 打开微信开发者工具，导入本目录。
2. 把 `project.config.json` 里的 `appid` 从 `touristappid` 替换成你自己的微信小程序 AppID。
3. 在微信公众平台配置小程序名称「半糖孕册」、类目、头像、服务内容和隐私保护指引。
4. 如果需要真实手机号验证码、云端反馈管理、跨设备同步，需要先接入微信云开发或自有后端。
5. 使用微信开发者工具点击「上传」，再到微信公众平台提交审核。
